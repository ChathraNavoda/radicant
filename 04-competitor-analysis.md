# Competitor Analysis

| Product | Strength | Weakness | Opportunity for Radicant |
|----------|----------|----------|---------------------------|
| GitHub | Code history | Doesn't capture reasoning | Preserve engineering decisions |
| Notion | Flexible documentation | Manual and disconnected | Automate engineering knowledge |
| Obsidian | Knowledge graph | Generic | Engineer-specific workflows |
| Capacities | Personal knowledge | Not developer focused | Engineering memory |
| Logseq | Graph thinking | Steep learning curve | Simplicity |
| LinkedIn | Career showcase | Static | Living engineering journey |
| ChatGPT | Problem solving | Knowledge isn't organized over time | Long-term engineering memory |

## Observation

Every existing product solves part of the problem.

None preserve an engineer's complete learning journey.
