# ADR-0001: Radicant is an Engineering Memory Platform

## Status

Accepted

## Date

2026-07-14

## Context

During product discovery we evaluated several possible product identities:

- Developer Journal
- AI Notebook
- Second Brain
- Knowledge Base
- Career Tracker

Each captured part of the problem but failed to express the complete vision.

## Decision

Radicant will position itself as an Engineering Memory Platform.

The product exists to help software engineers capture, connect and cultivate the knowledge behind their work.

## Consequences

This decision affects future product direction.

Features should preserve engineering knowledge rather than simply store notes.

The primary value is helping engineers remember how they became great.

Features that don't reinforce this mission should be questioned before implementation.
